# 🎮 Madlibs Game

A simple Python command-line game that creates a fun story based on the words entered by the user.

## 🚀 Features

* Interactive user input
* Generates a unique story
* Built with Python

## ▶️ Run the Project

```bash
python main.py
```

## 💡 Example

```text
My name is Refat.
Today I went to an amazing zoo.
In an exhibit, I saw an elephant.
The elephant was huge and dancing.
I was excited.
```
